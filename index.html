Quick Start

  1. Download and extract Printd for your platform.
  2. On Windows, download PDFtoPrinter.exe or SumatraPDF.exe and place it in the same directory as printd.exe.
  3. On Linux, ensure CUPS is installed and your printer is configured.
  4. Download the Printd configuration file from your contest on Toph and place it in the same directory as the Printd binary.
  5. Run Printd.

For detailed instructions, configuration options, and troubleshooting, see the README.
